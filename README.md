## 📖 Project Overview
This project focuses on analyzing an e-commerce dataset using Microsoft Excel.  
The goal is to clean raw data, perform analysis, and extract meaningful insights to support business decision-making.

---

## 🎯 Objectives
- Clean and preprocess raw dataset  
- Perform data analysis using Excel formulas  
- Identify trends and patterns in sales  
- Generate actionable business insights  

---

## 🛠 Tools & Technologies
- 📊 Microsoft Excel  
- 📈 Data Analysis Techniques  
- 🧹 Data Cleaning Methods  

---

## 📂 Dataset Description
The dataset contains:
- Customer information  
- Product details  
- Order data  
- Sales transactions  

---

## ⚙️ Methodology
1. **Data Cleaning**
   - Removed duplicates  
   - Handled missing values  
   - Corrected formatting  

2. **Data Transformation**
   - Structured tables  
   - Organized columns  

3. **Data Analysis**
   - Used formulas like `SUM`, `IF`, `XLOOKUP`  
   - Created summary tables  

4. **Visualization**
   - Charts and dashboards (if included)

## 📈 Key Insights
- 🏆 Top-performing products identified  
- 🌍 Regions with highest sales detected  
- 🛍 Customer purchasing behavior analyzed  
- 📅 Sales trends over time observed
